## Overview

This repository is a comprehensive Cyber Threat Intelligence (CTI) resource on **VECT Ransomware**. It provides structured intelligence analysis, malware behavior analysis, Indicators of Compromise (IOCs), MITRE ATT&CK mapping, YARA rules, and actionable recommendations.  

Designed for cybersecurity professionals and researchers, the content supports **threat hunting**, **incident response**, and **proactive defense strategies**.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Repository Structure

- **Cyber Threat Intelligence Report** – Main CTI report in PDF/Word formats.  
- **Diamond Model** – Analysis of adversary tactics and attack patterns.  
- **Malware Analysis** – Detailed behavior analysis of VECT ransomware samples.  
- **MITRE ATT&CK®** – Mapping of observed techniques to MITRE framework.  
- **Ransomware Tracker** – Timeline and tracking of VECT ransomware activity.  
- **VECT-Sample & IOCs** – Hashes, file metadata, and other indicators.  
- **YARA Rules** – Detection rules for hunting VECT ransomware.
